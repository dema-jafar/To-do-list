# To-Do List App

A simple and interactive To-Do List app built using HTML, CSS, and JavaScript.

Features

- Add new tasks
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Save tasks in browser using `localStorage`

---

Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API
